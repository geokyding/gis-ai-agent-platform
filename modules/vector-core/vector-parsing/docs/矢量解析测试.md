## ✅ 2. JSON（Postman / curl）

```
curl.exe -X POST http://localhost:8081/vector/shp/parse ^
  -H "Content-Type: application/json" ^
  -d "{\"path\":\"D:/works/temp/data/test_shp.shp\"}"
```

✔ 会走：

```
parseJson()
```

## ✅ 3. 浏览器 fetch

```
fetch('/vector/shp/parse', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    path: 'D:/works/temp/data/test_shp.shp'
  })
})
```
