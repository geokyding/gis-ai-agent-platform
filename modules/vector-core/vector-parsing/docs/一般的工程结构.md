# 一般的工程结构

```
com.xxx.project
│
├── controller        // 接口层
├── service           // 业务层
├── repository        // 数据访问层
├── dto               // ⭐ DTO（请求/响应）
│   ├── request
│   │    └── ShpParseRequest.java
│   └── response
│        └── ShpParseResponse.java
├── entity            // 数据库实体
├── config            // 配置
```
